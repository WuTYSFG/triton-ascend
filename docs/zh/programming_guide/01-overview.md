# 第一章 概述

## 背景

本文着重介绍在 NPU 上进行 Triton 算子开发中值得注意的问题，分为三个方面：

1. **多核任务并行**：介绍设置最大硬件核数的依据以及具体的实现
2. **单核数据搬运**：具体描述如何设置合适的循环内数据分块大小，介绍过程中常用的优化手段，并补充可能面临的 UB OVERFLOW 问题的处理方式
3. **单核数据运算**：回归单个算子，在单核数据运算层面着重介绍如何开发 Triton 算子，并强调相关的关键点
